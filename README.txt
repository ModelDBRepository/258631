Directory for paranoia as a deficit in non-social belief updating

Reed EJ, Uddenberg S, Suthaharan P, Mathys CD, Taylor JR, Groman SM, Corlett PR (2020) 

*Email: philip.corlett@yale.edu

Model fit to human and rodent data showing effects of paranoia and methamphetamine on behavior and model parameters

The .xlsx files contain clinical data for the three groups - online (MTurk), in-lab (IRL), and rat.

The .mat files contain the behavioral task data used as input into the HGF model fit

MATERIALS

The Hierarchical Gaussian Filter (HGF) toolbox v5.3.1 is freely available for download in the TAPAS package at https://translationalneuromodeling.github.io/tapas. We installed and ran the package in MATLAB and Statistics Toolbox Release 2016a (MathWorks, Natick, MA).
